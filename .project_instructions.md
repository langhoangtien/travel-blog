# Reiseblog - Travel Blog Project

## Overview

Production-ready travel blog with admin panel (Vietnamese UI) and public site (German UI).

## Tech Stack

- Next.js 14 (App Router), TypeScript, PostgreSQL, Prisma ORM, Tiptap Editor, Tailwind CSS
- S3 cloud storage for images
- NextAuth.js v4 for authentication

## Design Decisions

- **Admin panel**: All labels, buttons, validation in Vietnamese
- **Public site**: All UI text and content in German
- **Color**: Purple primary (#7C3AED / hsl 262 83% 58%)
- **Auth**: Email/password, no public registration, session-based JWT
- **Content**: Stored as HTML from Tiptap editor
- **Media**: Tracked in Media table, S3 storage, cleanup on delete/update
- **Roles**: ADMIN, EDITOR, AUTHOR (phase 1: no permission enforcement)

## Key Routes

- `/` - Public homepage (German)
- `/posts/[slug]` - Post detail (German)
- `/admin/login` - Login (Vietnamese)
- `/admin/posts` - Post management (Vietnamese)
- `/admin/posts/new` - Create post

## Phase 2 (Completed)

- **Tags system**: Many-to-many with posts via PostTag join table. Tag CRUD at `/admin/tags`. Chip-style TagInput with autocomplete + create-on-the-fly in post form.
- **Category CRUD**: `/admin/categories` with inline modal forms
- **User management**: `/admin/users` (ADMIN only) - CRUD with role assignment, password reset, active/inactive toggle
- **Admin Dashboard**: `/admin` with stat cards (posts, categories, tags, users) + recent posts
- **Posts upgrade**: Pagination, search, filters (status, category, author), sortable columns, bulk actions (publish/draft/delete/move category)
- **Public pages**: `/category/[slug]`, `/tag/[slug]`, `/autor/[slug]` (all in German)
- **Public components**: Premium travel magazine redesign with glassmorphism header (Startseite, Reiseziele dropdown, Über uns, Kontakt), search with click-outside/ESC, mobile drawer. Full-width hero image, featured/latest/popular posts sections, category cards with emoji icons. 4-column footer with categories + recent posts via `lib/footer-data.ts`. Article page: full-width cover hero, card-wrapped content, next/prev nav. About page `/ueber-uns`, Contact page `/kontakt` with form + `/api/contact`. Post cards have 3 variants (default, featured, horizontal). Max-w-7xl, rounded-2xl throughout, framer-motion animations

## Phase 4: Production Deployment

- **Dockerfile**: Multi-stage build (deps → builder → runner) at `nextjs_space/Dockerfile`, non-root user (nextjs:1001), healthcheck, standalone output
- **Docker Compose**: `docker-compose.production.yml` at project root - PostgreSQL 16 Alpine + Next.js app, internal network, no DB port exposed
- **Nginx**: `deploy/nginx/nerovia.conf` - reverse proxy, SSL, gzip, rate limiting, security headers, static cache
- **CI/CD**: `.github/workflows/deploy.yml` - GitHub Actions, ghcr.io registry, SSH deploy to VPS
- **Scripts**: `deploy/scripts/deploy.sh`, `rollback.sh`, `migrate.sh` for manual operations
- **Guide**: `deploy/DEPLOY_GUIDE.md` - complete VPS setup guide in Vietnamese
- **Prisma binaryTargets**: Added `linux-musl-openssl-3.0.x` for Docker Alpine x86_64
- **Sitemap**: Updated to include categories, tags, authors
- **Seed data**: 12 sample tags (Europa, Asien, Strand, Wandern, etc.) connected to posts
